# Termux-Theme
Script untuk mempercantik tampilan awal termux kamu !

<details open><summary><code>Perintah Script?</code></summary>

```php
$ git clone https://github.com/Sxp-ID/termux-theme
$ cd termux-theme
$ make install
$ ./main

Atau bisa juga run script nya dg ketik perintah
$ make run
```
